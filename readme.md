# cpp-basics

Personal C++ files I'm using to learn DSA. No project — just syncing my practice code.

## 📁 Structure

```
cpp-basics/
├── patterns/
│   └── pattern.cpp       # 22 patterns (stars, pyramids, diamonds, etc.)
└── stl/
    └── ...               # STL concepts and practice
```

## 🧠 What's Inside

### `patterns/`
A single file with 22 different patterns — triangles, pyramids, inverted shapes, diamonds, number patterns, and more. Good for getting comfortable with nested loops.

### `stl/`
Practice with the C++ Standard Template Library — vectors, maps, sets, stacks, queues, and other containers/algorithms.

## 🚀 Run Anything

```bash
g++ filename.cpp -o out && ./out
```
